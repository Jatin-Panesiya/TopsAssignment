Q :- 2 ] How to use JavaScript Output method?

Answer :-

JavaScript provides several methods for outputting data to various destinations, such as the browser's console, HTML document, or alert dialogs.

Here are some commonly used JavaScript output methods:

console.log(): This method is used to output messages to the browser's console, which is a debugging tool. It is commonly used for testing and debugging purposes. For Example:-

>> console.log("Hello, world!");

document.write(): This method is used to write content directly into the HTML document. It is simple to use but has limited use cases, as it overwrites the entire document when used after the page has finished loading. For example:-

>> document.write("Hello, world!");

innerHTML: This property is used to set the HTML content of an element. It allows you to dynamically update the content of an HTML element. For example:-

>> document.getElementById("myElement").innerHTML = "Hello, world!";

alert(): This method displays a dialog box with a message and an OK button. It is often used to provide simple notifications or display messages to the user. For example:

>> alert("Hello, world!");

prompt(): This method displays a dialog box that prompts the user for input. It is commonly used to get user input or retrieve values from the user. For example:

>> var name = prompt("Please enter your name:");

confirm(): This method displays a dialog box with a message and OK and Cancel buttons. It is used to get user confirmation or make simple yes/no decisions. For example:

>> var result = confirm("Are you sure you want to delete this item?");
